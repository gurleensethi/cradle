# Project: Cradle

## Overview

Cradle is a Go-based CLI tool that helps developers manage their local projects efficiently. It provides both a TUI (Terminal User Interface) and command-line commands for project management.

## Golang Instructions
- Follow idiomatic go, not idiotic go!
- Every function should be documented properly and concisely with what it does, focus on the documenting the behaviour of the function instead of implementation details, the documentation should be meaningful to the developer reading it.
